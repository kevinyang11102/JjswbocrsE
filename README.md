# 药用植物病虫害管理平台 java873

## 项目概述

药用植物病虫害管理平台旨在为农林科研机构和药用植物种植基地提供一个功能全面的信息管理解决方案。平台采用当前流行的Springboot后端和Vue前端技术开发，确保了系统的稳定性和用户界面的友好性。

## 技术栈

- 后端：Springboot
- 前端：Vue
- 数据库：设计合理的数据库结构，包含药用植物、病虫害种类、图片等详细信息

## 功能模块

- 药用植物信息查询
- 病虫害信息查询
- 图片资料管理
- 信息添加与修改

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 后端运行环境：Java 1.8+
- 前端运行环境：现代浏览器

## 部署步骤

1. 准备Java运行环境
2. 部署数据库服务器
3. 部署后端服务
4. 部署前端服务
